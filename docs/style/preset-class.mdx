> 帮助实现风格一致的 UI

## 布局系统 layout

规则:

- layout + 对应的布局属性的缩写 + row- + 一份的占用的大小, 例如
- layout 为一行，分为 20 份，一个 row-1 占用一份
- 控制布局的行为使用缩写，例如 __justify-content: center === j-c-c__

.layout 的默认属性

```css
display: flex;
flex-direction: row;
flex-wrap: wrap;
align-content: flex-start;
align-items: flex-start;
justify-content: flex-start;
```

## 内外边距

通用样式 margin, padding, 规则为边距类型缩写 + 以 5 px为单位，从 5px - 20px 的 class，例如

<Playground>
  <div style={{width: 300, height: 300, background: '#EEE', padding: '0.1px'}}>
    <div style={{width: 50, height: 50, background: '#DDD'}} className="mb10"></div>
    <div style={{width: 50, height: 50, background: '#DDD'}} className="ml10"></div>
  </div>
</Playground>

```css static
# mt10
margin-top: 10px;

# mu10 -> margin upset, 上下边距
margin: 10px 0;

# ms10 -> margin side, 左右边距
margin: 0 10px;

# pt10
padding-top: 10px;

# pu10 -> padding upset, 上下边距
padding: 10px 0;

# ps10 -> padding side, 左右边距
padding: 0 10px;
```
