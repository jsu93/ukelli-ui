## 自定义表单组件

通过 type = 'customForm' 来实现，实现 getCustomFormControl 接口

```js
//getCustomFormControl 会尝试实例化 component 组件，并传入 onChange 接口，可以调整表单的 ref 的值
const formOptions = [
  {
    ref: 'customer',
    type: 'customForm',
    getCustomFormControl: () => {
      return {
        component: 'div',
        props: {}
      }
    },
    required: true,
    title: '自定义的表单组件',
    values: {
      value1: 'value1',
      value2: 'value2',
      value3: 'value3',
    }
  }
]
```
