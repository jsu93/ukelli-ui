# Toast

> Toast 需要放到 position != static 的容器之中, 通过 ref 来让 Taost 显示

<PropsTable of={Toast} />

<Playground>
  {
    () => {
      let toast;
      return (
        <div className="relative">
          <div style={{height: 100}}></div>
          <span className="btn theme" onClick={e => {
            toast.show('打开了一个 Toast', 'green')
          }}>开启一个 taost</span>
          <Toast ref={e => toast = e}/>
        </div>
      )
    }
  }
</Playground>
